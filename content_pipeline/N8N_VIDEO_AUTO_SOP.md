# 🤖 n8n 全自動化短影音生產線 (Faceless Video Pipeline)

如果完全不露臉，且依賴 n8n 作為自動化中樞，我們可以打造一條 **「從文字到 MP4 影片」的無人化流水線**。

## 📍 系統架構 (API 串接閉環)

這套系統需要 n8n 串接多個第三方 AI 工具的 API，流程如下：

### 1. 觸發與腳本生成 (Trigger & Script)

- **工具：** Notion (Idea Inbox) + OpenAI (GPT-4) API
- **n8n 節點動作：**
  1. 定時（例如每週一、四）從 Notion 抓取一則新的「市場洞察」。
  2. 呼叫 OpenAI API，將該洞察改寫為 45 秒的短影音「旁白逐字稿 (Voiceover)」。

### 2. 語音生成 (Text-to-Speech)

- **工具：** ElevenLabs API (業界最逼真的人聲)
- **n8n 節點動作：**
  1. 將生成的逐字稿送入 ElevenLabs。
  2. 選擇一個具備權威感/療癒感的專屬聲音模型 (Voice ID)。
  3. 接收生成的 `.mp3` 音檔。

### 3. 視覺與影片合成 (Video Generation)

- _這是最困難的一步，因為單純的圖文拼接在短影音時代流量極差。_
- **工具：** HeyGen API (AI 虛擬人) 或 Creatomate API / JSON2Video API (自動化剪輯)
- **n8n 節點動作 (方案 A - 虛擬分身)：**
  如果你事先錄製好一個「你的 AI 數位分身 (Avatar)」，n8n 可以把 ElevenLabs 的音檔丟給 HeyGen，自動生成「你」在講話的影片。
- **n8n 節點動作 (方案 B - 素材拼接)：**
  使用 Creatomate，傳遞預設好的模板（背景播放 TimeWaver 數據庫截圖或抽象頻率波紋，加上自動動態字幕），然後把音檔合上去，輸出 `.mp4`。

### 4. 自動發布或排程 (Distribution)

- **工具：** Instagram Graph API / YouTube Data API / Buffer
- **n8n 節點動作：**
  將生成的 `.mp4` 影片與 GPT 生成的貼文文案，直接排程或自動推送到社群平台。

## ⚖️ 丞相的戰略評估 (優缺點分析)

### 優勢 (Pros)

- **絕對的規模化：** 你完全不需要開相機、不需面對鏡頭，可以做到日更（Daily Post）。
- **徹底的「睡後行銷」：** 整個漏斗從「產影片 -> 發影片 -> 拿名單 -> 寄音頻」全部交給伺服器處理。

### 劣勢 (Cons)

- **API 成本極高：** ElevenLabs + HeyGen/Creatomate 的 API 費用，每月至少 $50 - $150 USD 起跳。
- **技術門檻：** n8n 處理 `.mp4` 這種大型二進制檔案（Binary Data）容易遇到記憶體溢出或 Timeout 的問題，維護成本較高。
- **轉換率致命傷：** 在「身心靈、療癒、高客單價檢測」的市場，如果消費者發現影片是 AI 虛擬人或純機器剪輯，**信任感會瞬間清零**。沒有信任，就不會有人填寫測驗、留下 Email，更別說付費買音頻。
